# Changelog

## v1.28.0

- 入力中の本文をチャット／プロファイル単位で自動保存する入力下書き機能を追加しました。
- Dropbox接続済みの場合、下書きをDropbox App Folderのdrafts配下へ共有保存するように対応しました。
- チャット再表示時、入力欄が空の場合のみ最新下書きを自動復元するようにしました。
- 送信成功後に該当下書きを削除扱いにし、古い下書きの誤復元を防止しました。
- アプリバージョンとキャッシュバージョンを1.28.0に更新しました。

## v1.27.6

- スマホ表示時の色設定UIで、操作部が右寄せになるよう調整しました。
- チャット左上部の同期ボタンを、設定画面への移動ではなく即時同期実行に変更しました。

## v1.27.5

- ヘッダー内ボタンのマウスオーバー/タップ時の視認性を改善しました。
- 暗いヘッダー色でもヘッダー内ボタンのhover/tap時に見やすくなるよう調整しました。
- テーマ設定内の「色」見出しを「色設定」に変更しました。
- バージョン更新時の更新内容を、最初から表示してOKで閉じる形式に変更しました。

## v1.27.4

- PC表示時ワイドモード設定をテーマ設定セクションへ移動しました。
- アクセントカラー方式と個別設定方式を切り替えられるようにしました。
- テーマ色の個別設定UIを、選択中の方式に応じて有効/無効化するようにしました。
- ユーザーメッセージ表記へ統一しました。

## v1.27.3

- テーマ設定を独立したセクションに整理しました。
- ダークテーマ設定をテーマ設定セクションへ移動しました。
- アクセントカラーから主要なテーマ色を調整できる設定を追加しました。
- ユーザーメッセージの色設定を追加しました。

## v1.27.2

- ヘッダー文字色と新規チャットボタン色が反映されない問題を修正しました。
- ヘッダー文字色と新規チャットボタン色にリセットボタンを追加しました。
- 色設定エリアのUIを整理しました。

## v1.27.1

- ヘッダー文字色を自動/手動で選べるようにしました。
- 新規チャットボタン色を設定できるようにしました。
- ヘッダー内アイコンやボタンの文字色追従を改善しました。

## v1.27.0

- PWAの見た目、テーマ色、バージョン表示を整理
- ヘッダー背景色に応じてヘッダー文字色を自動調整
- `meta theme-color` をヘッダー色へ追従
- 設定画面にアプリバージョンとリリースノート表示を追加
- 一部ブラウザのPWAでは、OS側ステータスバー色が `theme-color` に完全追従しない場合があります。
- PWAキャッシュを v1.27 に更新

## v1.26.0

- PWAのedge-to-edge表示を改善
- `viewport-fit=cover` と safe area 対応を追加
- ヘッダーと下部入力欄のステータスバー/ホームバー被りを軽減
- GitHub Pages / PWAキャッシュを v1.26 に更新
